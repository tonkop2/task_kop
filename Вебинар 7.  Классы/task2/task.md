<h2>Задача 2</h2>
Напишите класс <code><span class="colortext">PersonInfo</span></code>.
<br>
Экземпляр класса создается из следующих атрибутов:
<ol>
<li>Строка - "Имя Фамилия";</li>
<li>Число - возраст сотрудника;</li>
<li>Подразделения - от головного до того, где работает сотрудник.</li>
</ol>
<p>Реализуйте методы класса:
<ol>
<li><code>short_name</code>, который возвращает строку <code>Фамилия И.</code></li>
<li><code>path_deps</code> - возвращает путь <code>"Головное подразделение --> ... --> Конечное подразделение"</code></li>
<li><code>new_salary</code> - Директор решил проиндексировать зарплаты, и новая зарпалата теперь вычисляется по формуле:
<pre><code>1337*Возраст*суммарное кол-во вхождений трех наиболее часто встречающихся букв из списка подразделений</code></pre>
<b>Регистр букв имеет значение.</b> "А" и "а" - разные буквы.
</li>
</ol>
<br>
<p><b>Пример:</b> (Ввод --> Вывод)

<ol>
  <li><pre><code>PersonInfo('Александр Шленский',32, 'Разработка', 'УК', 'Автотесты').short_name() --> 'Шленский А.'</code></pre></li>
<li><pre><code>PersonInfo('Александр Шленский', 32, 'Разработка', 'УК', 'Автотесты').path_deps() --> 'Разработка --> УК --> Автотесты'</code></pre></li>
<li><pre><code># т.к. буква "т" встречается 4 раза, "а" - 3 раза, 'о' - 2 раза, остальные - по одной, 
# то сумма трёх самых частых букв равна 4+3+2 = 9.
# Итого по формуле 1337*32*9 = 385056
PersonInfo('Александр Шленский', 32, 'Разработка', 'УК', 'Автотесты').new_salary() --> 385056</code></pre></li>
</ol>
